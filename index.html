¿Cómo exporto mi calendario desde Google? →
  1. Entrá a calendar.google.com en una computadora.
  2. Arriba a la derecha: ⚙ Configuración → menú izquierdo Importar y exportar.
  3. Sección Exportar → botón Exportar. Se descarga un .zip con un .ics por cada calendario.
  4. Soltá ese .zip (o los .ics de adentro) en el recuadro de arriba.
¿Cómo obtengo el Client ID? (configuración única, ~10 min) →
  1. Andá a console.cloud.google.com y creá (o elegí) un proyecto.
  2. APIs y servicios → Biblioteca: buscá Google Calendar API y tocá Habilitar.
  3. Pantalla de consentimiento OAuth: tipo Externo. Completá nombre y tu email. En Usuarios de prueba agregá tu propia cuenta de Google.
  4. Credenciales → Crear credenciales → ID de cliente de OAuth → tipo Aplicación web.
  5. En Orígenes autorizados de JavaScript agregá exactamente: http://localhost:8000
  6. Copiá el Client ID que termina en .apps.googleusercontent.com y pegalo arriba.
La conexión en vivo necesita que abras esta página servida en http://localhost (ver instrucciones abajo o en el README). El método de archivo .ics funciona sin nada de esto.
